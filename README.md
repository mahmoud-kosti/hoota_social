## 🎯 نبذة تعريفية

**HOota Social** تطبيق ويب تفاعلي **SPA** مبني بـ **Vanilla JavaScript** + **Supabase BaaS** بدون أطر عمل خارجية.

تصميم **Glassmorphism 2026** مع تدرجات متحركة و `backdrop-filter: blur(15px)`. نظام أمان **Row Level Security** على مستوى PostgreSQL، وتجربة **Mobile-First** بسرعة < 1 ثانية.

تم التطوير بمساعدة **الذكاء الاصطناعي Pair Programming** لتطبيق أفضل الممارسات في الأمان والأداء والتصميم.
