# 🧠 HOota Social

> منصة تواصل اجتماعي حديثة بروح المستقبل ✨  
> تصميم احترافي + أداء عالي + تجربة مستخدم متطورة

---

## 👨‍💻 عني

أنا **محمود عبدالله**  
🎓 طالب في **جامعة النيل الأبيض – السودان**  
💻 تخصص **تقانة المعلومات (IT)**  

🚀 مهتم بـ:
- تطوير الويب
- الذكاء الاصطناعي
- بناء المشاريع التقنية

🔥 هدفي: إنشاء منصة تواصل سودانية تنافس عالميًا

---

## ⚡ فكرة المشروع

**HOota Social** هو تطبيق تواصل اجتماعي حديث يركز على:

- ⚡ السرعة
- 🎯 البساطة
- ✨ الجمال
- 📱 تجربة المستخدم

📌 المشروع حاليًا **قيد التطوير** وسيتم تحسينه وإضافة مميزات قوية قريبًا

---

## ✨ المميزات الحالية

- 🎨 تصميم Glassmorphism 2026
- 📱 Responsive (Mobile First)
- 🔐 نظام تسجيل دخول باستخدام Supabase
- 🧠 بنية SPA (Single Page Application)
- ⚡ أداء سريع جدًا
- 🌙 دعم الوضع الليلي
- 🔥 واجهة حديثة مستوحاة من تطبيقات الذكاء الاصطناعي

---

## 🚀 المميزات القادمة

- 💬 نظام محادثات فورية (Realtime Chat)
- 🤖 توصيات باستخدام الذكاء الاصطناعي
- 🎥 دعم الفيديوهات والريلز
- 👥 نظام متابعة (Follow System)
- 🔔 إشعارات مباشرة
- 🛡️ تحسين الأمان (JWT + RLS)
- 📊 لوحة تحكم للمستخدم

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3 (Animations + Glassmorphism)
- JavaScript ES6+

### Backend
- Supabase
- PostgreSQL
- Authentication + RLS
